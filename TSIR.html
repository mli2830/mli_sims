Discrete Latent State TSIR Chain Binomial

Model

The Reed-Frost model is a discrete-time SIR model that assumes recovery one time step after infection. Let \(N\) be the population size, \(I_{1}\) be the initial number of infectives, \(\beta\) be the probability of adequate contact per infective, then the RF model can be written as:

\[ S_{1} = N - I_{1} \\ I_{t} \sim \textrm{Binomial}(\textrm{prob} = 1-(1-\beta)^{I_{t-1}} , \textrm{size}=S_{t-1}) \\ S_{t} = S_{t-1} - I_{t} \]

We modified the RF model as a latent binomial process and added an observation process with reporting probability \(r\) and assumed \(N_{e}\) is from an effective population with an effective proportion \(\alpha\) of the upper limit population size N. The modified RF model can be written as:

\[ N_{e} \sim \textrm{Binomial}(\textrm{prob} = \alpha, \textrm{size}=N) \\ S_{1} = N_{e} - I_{1} \\ I_{t} \sim \textrm{Binomial}(\textrm{prob} = 1-(1-\beta)^{I_{t-1}} , \textrm{size}=S_{t-1}) \\ Obs_{t} \sim \textrm{Binomial}(\textrm{prob}=r,\textrm{size}=I_{t}) \\ S_{t} = S_{t-1} - I_{t} \]

Hybrid model

The orginal idea of hybridization was to improve the mixing for r with some sort of Scaling trick/ cliff. We later realized we can take this even furtherand use a more powerful MCMC algorithm (Hamiltonian Monte Carlo). For now, let’s try to figure out the problem in JAGS

First we want to change the discrete latent process to continuous by setting the first and second moment (moment-matching (MM)) of the binomial distribution to a continuous distribution (ie. gamma in this case) and observation process to a poisson process (more detail on why possion). \ … skip all the moment-matching algebra … \ We can rewrite the model (todo: reference equation/system of equations?) as:
